# Visibility Modes

The visibility mode controls what household members can see when logged in. It's set when you create your household and can be changed later.

## The four modes

### Fully Shared

> "We see everything together."

All household members see all accounts and transactions. No privacy between members. The simplest mode for couples or families with fully combined finances.

**Good for:** couples who share everything, individuals, single-person deployments.

### Separate with Joint View

> "Individual budgets, shared overview."

Each member has a private view of their own accounts. A joint view aggregates everything for members who want the full picture. Individual budgets run separately while a shared overview is available.

**Good for:** couples who maintain some financial independence but want visibility into the household totals.

### Role-Based

> "Owner sees all, members see their own."

The Household Owner sees all accounts and transactions across all members. Members see only what's attributed to them. Useful when one person manages household finances and others are participants with limited visibility.

**Good for:** households with one financial manager and multiple participants (e.g., adult children, or one partner who handles finances for both).

### Admin-Controlled

> "Fully custom."

The Household Owner explicitly grants per-member access. Each member's visibility is individually configured. Most flexible but most effort to set up and maintain.

**Good for:** complex households with specific requirements, or when you need fine-grained control that the other modes don't provide.

## How to choose

```
Do you want all members to see everything?
├── Yes → Fully Shared
└── No
    ├── Does one person manage finances for everyone?
    │   ├── Yes → Role-Based
    │   └── No
    │       ├── Do members want full individual privacy?
    │       │   ├── Yes → Separate with Joint View
    │       │   └── Custom requirements → Admin-Controlled
```

## Changing visibility mode later

You can change the visibility mode at any time in **Settings → Household**. The change takes effect immediately. No data is deleted when you change modes — it's a view permission change, not a data change.
